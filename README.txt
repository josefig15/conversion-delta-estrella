El programa DELTA ESTRELLA V0_1.txt ha sido escrito en TI-Basic compatible con calculadoras TI-83/84.

También se incluye una versión del programa compatible para OCTAVE.
Para usarlo en octave diríjase al directorio dónde está el script delta_estrella.m y de ejecutar.

Se recomienda Abrir el archivo DELTA ESTRELLA V0_1.txt en Notepad++ 

Funcionamiento:
	- El programa pregunta el tipo de Conversion
		Si oprime numero 1 hace conversion Delta a Estrella.
		Si oprime numero 2 hace conversion Estrella a Delta.
	
	- El programa pide que ingrese los valores de las resistencias del arreglo.
		Estos datos se almacenan en Vector L1.
	
	- El programa entonces calcula la respectiva conversion y muestra mesajes de las ubicaciones de los resistencias.
	
	- El resultado de la conversion queda almacenado en vector L2.
