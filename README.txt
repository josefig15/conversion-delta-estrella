El Prograna hace la conversion entre arreglos de resistores en Delta a Estrella y visceversa.

El archivo "DELTA ESTRELLA V0_1.txt" ha sido escrito en TI-BASIC compatible con calculadoras TI-83/84.

El archivo "delta_estrella.m" ha sido escrito en OCTAVE.

Como usar: 
- Ejecutar el programa en la calculadora o Ejecutar el script en Octave.
- El programa pregunta el tipo de Conversion:
    Si oprime Numero 1 hace conversion Delta a Estrella.
    Si oprime Numero 2 hace conversion Estrella a Delta.   (o Cualquier numero diferente de 1).
- El programa pide que ingrese los valores de las resistencias del arreglo. Estos datos se almacenan en Vector L1.
- El programa calcula la respectiva conversion y muestra mesajes de las ubicaciones de los resistencias.
- El resultado de la conversion queda almacenado en vector L2.

Se recomienda Abrir el archivo DELTA ESTRELLA V0_1.txt en Notepad++
